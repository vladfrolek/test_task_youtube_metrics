# YouTube Metrics Reports

CLI-приложение для обработки CSV-файлов со статистикой YouTube-видео.

Приложение читает один или несколько CSV-файлов с метриками видео и формирует отчёт `clickbait`. В отчёт попадают видео с высоким CTR и низким удержанием зрителей.

## Что делает приложение

Приложение находит кликбейтные видео по условиям:

- `ctr > 15`
- `retention_rate < 40`

В итоговом отчёте выводятся только колонки:

- `title`
- `ctr`
- `retention_rate`

Результат сортируется по убыванию `ctr`.


Архитектура приложения позволяет добавлять новые отчёты через `report_registry.py`.


## Зависимости

- `tabulate`

Для тестов используется:
