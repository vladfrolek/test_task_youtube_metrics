# YouTube Metrics Reports

CLI-приложение для обработки CSV-файлов со статистикой YouTube-видео.

Приложение читает один или несколько CSV-файлов с метриками видео и формирует отчёт `clickbait`. В отчёт попадают видео с высоким CTR и низким удержанием зрителей.

## Что делает приложение

Приложение находит кликбейтные видео по условиям:

- `ctr > 15`
- `retention_rate < 40`

В итоговом отчёте выводятся только колонки:

- `title`
- `ctr`
- `retention_rate`

Результат сортируется по убыванию `ctr`.

## Пример запуска

Запуск с одним CSV-файлом:

```bash
python -m project.cli --files files/stats1.csv --report clickbait
```

Запуск с несколькими CSV-файлами:

```bash
python -m project.cli --files files/stats1.csv files/stats2.csv --report clickbait
```

## Поддерживаемые отчёты

Сейчас поддерживается один отчёт:

```text
clickbait
```

Архитектура приложения позволяет добавлять новые отчёты через `report_registry.py`.


## Как работает приложение

1. Пользователь запускает приложение и передаёт пути к CSV-файлам через `--files`.
2. Через `--report` пользователь указывает название отчёта.
3. Приложение проверяет, существует ли такой отчёт.
4. CSV-файлы читаются и объединяются в общий список видео.
5. Для отчёта `clickbait` выбираются видео с `ctr > 15` и `retention_rate < 40`.
6. Результат сортируется по убыванию `ctr`.
7. Итоговый отчёт выводится в консоль в виде таблицы.
